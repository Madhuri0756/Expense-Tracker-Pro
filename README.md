# 💰 Expense Tracker Pro

A modern and responsive Expense Tracker web application built using HTML, CSS, and JavaScript.

## 🚀 Features

- Add, edit, and delete expenses
- Search expenses
- Filter by category
- Filter by tag
- Dashboard with analytics
- Category-wise spending visualization
- LocalStorage support
- CSV export
- Responsive design
- Pagination for expense records

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- LocalStorage API

## ▶️ How to Run

1. Download or clone the repository.
2. Open `index.html` in any modern web browser.
3. Start adding your expenses.

No installation or additional software is required.
